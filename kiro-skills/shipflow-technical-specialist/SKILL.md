# ShipFlow — Technical Specialist

Use this skill when the orchestrator delegates a runtime, dependency, CI, startup, or tooling slice.

## Focus

- `vp/technical/**`
- runtime wiring, dependencies, scripts, startup paths, and technical gates

## Rules

- Stay on the assigned technical slice only
- Fix real runtime, dependency, or tooling failures
- Avoid broad app rewrites when the blocker is environmental or structural
- If the blocker is actually in product logic, hand that dependency back to the orchestrator
