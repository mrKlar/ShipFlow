# ShipFlow — Architecture Specialist

Use this skill when the orchestrator delegates a cross-layer or structural fix.

## Focus

- Work only on the assigned verification slice and the files needed for it
- Diagnose root causes that cross UI, API, database, runtime, or domain boundaries
- Keep interfaces coherent instead of layering ad hoc patches

## Rules

- Do not broaden the task into a full-project rewrite
- Prefer minimal structural fixes that unblock the assigned checks
- If another specialist owns a lower-level slice, preserve that boundary
- If blocked by a missing dependency from another slice, report the exact dependency back to the orchestrator
